# The InGen Diaries

![The InGen Diaries](https://raw.githubusercontent.com/00Ankit00/The-InGen-Diaries/master/public/fossil_bg.jpg)

**The InGen Diaries** is a highly immersive, interactive cinematic reference web application designed as an archive of prehistoric and synthesized lifeforms. It offers a dual-themed experience that juxtaposes the sterile, corporate "InGen" aesthetic with a rugged "Fossil" aesthetic.

## 🦖 Features

- **Dual Modes (Fossil & InGen):** Experience the archive through two distinct visual themes. Choose between the dark, classified corporate terminal interface or the glowing, prehistoric fossil record view.
- **Comprehensive Roster:** Browse an extensive list of 37 assets, complete with high-quality imagery, scientific classifications, danger levels, dietary profiles, and rich lore details.
- **Cinematic UI/UX:** Built with modern web technologies and enhanced with GSAP animations, dynamic backgrounds, CRT scanlines, and audio effects to emulate an authentic Jurassic Park/World terminal.
- **Interactive Map:** Explore the geography of the islands with a sleek, interactive map interface.
- **Hybrid Genome Lab:** A dedicated secure section detailing classified genetic combinations.
- **Real vs Reel:** Compare the cinematic representations of the creatures with their actual paleontological records.

## 💻 Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [GSAP](https://gsap.com/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Data Management:** Static TypeScript data structure with robust interfaces.

## 🚀 Getting Started

To run this project locally, follow these steps:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/00Ankit00/The-InGen-Diaries.git
   cd The-InGen-Diaries
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Explore the Archive:**
   Open [http://localhost:3000](http://localhost:3000) with your browser to access the system.

## 📂 Project Structure

- `src/app/` - The core application pages (Selection, Roster, Map, Lab, Timeline, Accuracy).
- `src/components/` - Reusable UI components (SiteHeader, ThreatLevel, SpeciesCard, AudioManager).
- `src/data/` - The comprehensive static data structure detailing all creatures and lore.
- `public/` - Static assets including audio files, images, and backgrounds.
